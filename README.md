# Objectif de la formation en développement
Créer une page d’authentification sur un projet de gestion des photos

Tout particulièrement faire un zoom sur l'utilisation des 'User' de Django pour :
- Créer une page d'authentification
- Gérer des droits différents sur les objets traités

# Le projet support
"Un collectif de photographes cherche un moyen de montrer et de partager son travail à un large public.
Ils veulent pouvoir mettre leurs photos en ligne et créer des posts à leur sujet sur un blog.
Ils ont fait appel à vous en tant que développeur Django, et 
vous ont demandé de créer une application web qui leur permette justement de faire cela. 
Ils ont besoin d’avoir deux types d’utilisateurs, les abonnés et les créateurs.
Seuls les créateurs doivent pouvoir créer du contenu.
Ce contenu doit ensuite être partagé dans un flux social, et
les abonnés doivent pouvoir choisir quels créateurs ils veulent suivre."

### Un assistant IA
Dès que je serai sûr de la totalité du code que je produis, j'installerai un assistant IA pour démultiplier la production de code dans un contexte de qualité de code sûr.

L'outil utilisera des produits OpenSource et locaux : 
- Ollama,
- Codestral, l'outil de Mistral AI spécialisé dans le code,
- Un connecteur de l'IA local à Codium (utiliser Continue??)

L'IA sera utilisé dans : 
- la production de code,
- la production de la documentation du code,
- le contrôle qualité du code par :
  - une assistance à la création des tests, et
  - (si possible) des audits réguliers du code produit, conforme aux normes en la matière.
